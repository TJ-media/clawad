제거 안내

광고 표시를 멈추거나, 클라이언트를 제거하거나, 계정을 삭제하는 방법을 안내합니다.

클로애드 이용은 전적으로 이용자의 선택입니다. 이용자는 언제든지 광고 표시를 멈추거나 서비스를 완전히 떠날 수 있으며, 그로 인한 불이익은 없습니다. 아래에서 상황에 맞는 방법을 선택하시기 바랍니다.

어떤 것을 선택해야 하나요?
  • 잠시만 광고를 끄고 싶다 → 1. 광고 일시중지
  • 클라이언트를 지우되 계정과 리워드는 남기고 싶다 → 2. 클라이언트 제거
  • 서비스를 완전히 떠나고 데이터도 지우고 싶다 → 3. 계정 삭제(탈퇴)

1. 광고 일시중지

계정과 적립된 리워드를 유지한 채 광고 표시만 멈춥니다. 일시중지 중에는 광고가 표시되지 않으므로 리워드도 적립되지 않습니다.

npm run clawad:pause

일시중지 중에도 이미 확정된 리워드는 그대로 유지되며 교환할 수 있습니다. 다시 광고를 표시하려면 클라이언트 설정에서 일시중지를 해제하시면 됩니다.

2. 클라이언트 제거

개발 도구에서 클로애드 클라이언트를 완전히 제거합니다.

npm run clawad:uninstall

제거하면 어떻게 되나요?

  • 클로애드가 설치 시점에 백업해 둔 기존 상태줄(statusLine) 설정이 원상복구됩니다.
  • 광고 표시와 노출 정보의 자동 수집이 즉시 중단됩니다.
  • 기기에 저장된 로컬 캐시와 가명 기기식별자가 삭제됩니다.
  • 계정과 이미 확정된 리워드는 그대로 유지됩니다. 웹에서 로그인하여 리워드를 교환할 수 있습니다.

제거 후 기존 상태줄 설정이 정상적으로 복구되지 않은 경우 문의 창구로 알려주시면 도와드립니다.

3. 계정 삭제(탈퇴)

서비스 이용을 완전히 종료하고 회원 정보를 삭제합니다.

  1. 클로애드 웹에 로그인합니다.
  2. 계정 설정 화면으로 이동합니다.
  3. 탈퇴를 선택하고 안내에 따라 확인합니다.
탈퇴 전에 반드시 확인하세요
  • 교환하지 않은 리워드는 탈퇴와 동시에 소멸하며 환급되지 않습니다. 남은 리워드가 최소 교환 기준 이상이라면 탈퇴 전에 먼저 교환하시기 바랍니다.
  • 진행 중인 교환 건이 있는 경우, 상품 발송이 완료된 후 탈퇴하시는 것을 권장합니다.
  • 탈퇴 후에는 같은 소셜 계정으로 다시 가입할 수 있으나, 이전 계정의 리워드와 이용 내역은 복구되지 않습니다.

탈퇴하면 데이터는 어떻게 처리되나요?

데이터처리
소셜 로그인 식별자, 회원 식별자지체 없이 파기
상품 수령 이메일 주소지체 없이 파기(복구 불가)
등록된 기기 정보지체 없이 파기
광고 노출 기록·리워드 원장회원 식별자와의 연결을 끊어 개인을 알아볼 수 없도록 처리한 후, 통계·정산 목적으로 보관
동의 이력동의가 있었다는 사실의 증명을 위해 필요한 범위에서 보존

운영자가 관계 법령에 따라 일정 기간 보존해야 하는 정보는 해당 기간 동안 분리 보관 후 파기합니다. 자세한 사항은 개인정보처리방침 제3조를 참고하시기 바랍니다.

4. 탈퇴 없이 내 데이터만 확인·삭제하고 싶은 경우

계정을 유지하면서 본인의 데이터를 확인하거나 일부 처리의 정지를 요구할 수 있습니다.

  • 데이터 내보내기 — 계정 설정 화면에서 운영자가 보유한 본인 데이터를 파일로 내려받을 수 있습니다.
  • 기기 등록 해제 — 계정 설정 화면의 기기 목록에서 개별 기기를 해제할 수 있습니다.
  • 그 밖의 열람·정정·삭제·처리정지 요구 — 개인정보 문의를 통해 접수할 수 있습니다.

5. 서비스 이용에 동의하지 않는 경우

이용약관 또는 개인정보처리방침에 동의하지 않으시면 서비스에 가입하지 않으셔도 됩니다. 이미 가입하신 후 개정된 약관에 동의하지 않는 경우에는 위 3. 계정 삭제(탈퇴) 절차를 통해 이용계약을 해지하실 수 있습니다.

동의를 거부하시더라도 별도의 불이익은 없으며, 다만 서비스의 성격상 광고 표시와 리워드 적립 기능은 이용하실 수 없습니다.
