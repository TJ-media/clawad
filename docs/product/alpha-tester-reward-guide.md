# 알파 테스터 리워드·지급 안내 (CLAW-66)

폐쇄형 알파에서 리워드가 어떻게 쌓이고, 확정되고, 쿠폰으로 지급되는지 안내한다. 화면에 보이는 값과 실제 지급의 관계를 명확히 해 혼동을 줄인다.

> 리워드는 **비구매형·비양도형**입니다. 충전·양도·현금 환급이 없으며, 클로애드가 지정한 상품(모바일 쿠폰)으로만 교환됩니다. 화면의 수익 표시는 **예상**이며 서버 검증·정책으로 최종 확정됩니다.

---

## 1. 리워드 상태 용어

| 상태 | 뜻 | 화면 표기 |
| --- | --- | --- |
| **예상 적립** | 광고를 봤고 아직 서버 검증 전인 추정치 | "예상" |
| **검증 중** | 서버가 유효 노출·상한·부정 여부를 검증하는 중 | "검증 중" |
| **확정 리워드** | 검증을 통과해 교환에 쓸 수 있는 포인트 | "확정" |
| **교환 대기** | 상품 교환을 신청해 확정 포인트가 차감되고 발송을 기다리는 상태 | "신청됨" |
| **지급 완료** | 운영자가 쿠폰을 발송 완료한 상태 | "발송 완료" |
| **회수** | 부정·오류로 적립이 취소된 경우(원장 반대 분개로 정정) | — |

- 교환에는 **확정 리워드만** 쓸 수 있습니다. 검증 중 포인트로는 교환할 수 없습니다.
- 예상·검증 중 값은 검증 결과에 따라 줄어들 수 있습니다(부정·중복·상한 초과 등).

## 2. 지급(교환) 방법

1. 리워드 샵에서 상품을 고르고 교환을 신청합니다. 최소 교환 포인트 이상이어야 합니다(정책값, 화면에 표시됨).
2. 교환 시 **쿠폰을 받을 이메일**을 입력하고 수집·이용에 동의합니다. 이 주소로만 발송되며, 발송 후 파기됩니다. (로그인 계정과 별개이며, 로그인 식별자로 쓰지 않습니다.)
3. 알파 단계에서는 운영자가 **수동으로** 쿠폰을 발송합니다(영업일 기준 1~2일 소요).
4. 발송이 완료되면 상태가 "발송 완료"로 바뀝니다.

## 3. 실패·취소 시

- 발송 실패·취소 시 차감했던 확정 포인트는 **원복**됩니다. 교환 내역에서 상태를 확인할 수 있습니다.
- 재발송이 필요하면 포인트가 원복된 뒤 다시 교환을 신청하면 됩니다.

## 4. 개인정보

- 수집·이용하는 발송 이메일은 쿠폰 발송 목적에만 쓰이며, 발송 완료·취소·탈퇴 시 파기됩니다. 자세한 내용은 개인정보 처리방침을 참고하세요.
- 발송 대기 중 교환이 있으면 탈퇴가 일시 제한됩니다(발송 완료 또는 취소 후 탈퇴).

## 5. 세무 안내

지급에 대한 세무 처리 기준은 확정되지 않았습니다. 확정 시 별도 안내합니다. (본 안내는 세율·과세 기준을 확정 사실로 제시하지 않습니다.)
