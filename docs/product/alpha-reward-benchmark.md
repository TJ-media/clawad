# 알파 리워드 벤치마크·예산 분석

> 상태: 참고 분석. 실제 단가·예산은 세무(CLAW-13)·전자금융(CLAW-14) 결론과 함께 확정한다.
> 여기 수치는 사업 판단용 추정이며 투자·재무 자문이 아니다.

## 0. 목적

알파(1,000명 가정, 광고주에게 **무료로 광고 노출** = 매출 0) 시 회사가 부담할 리워드 규모와,
알파 테스터 리텐션에 실제로 필요한 리워드 구조를 벤치마크로 정리한다.

## 1. 현재 클로애드 정책값

`policy/reward-policy.default.json` (실험 기본값):

| 항목 | 값 |
|---|---|
| 인정 노출 1,000회당 | 300P (1P = 1원 상당 비현금 쿠폰) |
| 계정당 일일 노출 상한 | 500회 |
| 계정당 일일 리워드 상한 | 150P (= 150원/일) |
| 최소 교환 | 3,000P |
| 상한 기준 첫 교환 도달 | 20일 (3,000 / 150) |

노출은 15초 로테이션·5초 viewability 기준 → 활발한 대기시간 1시간 ≈ 최대 240회, 하지만 대기는
세션의 일부이므로 보수적으로 잡는다.

## 2. 경쟁·유사 서비스 벤치마크

### 2.1 kickbacks.ai (원본 아이디어, 클린룸 참고)

> 공개 마케팅 페이지 기준. 원본 소스코드는 열람하지 않았다(클린룸 원칙).

| 항목 | 내용 |
|---|---|
| 배분 | 광고 매출의 **50%를 개발자에게 (현금)** |
| 과금 단위 | 1블록 = 1,000회 5초 노출, 클릭 = 노출 50배 |
| 관측 단가 | 초기 $5 floor → 이후 블록당 serving floor $31, top bid $111 (변동 큼) |
| 출금 | $10 도달 시 현금 (론칭 초기엔 출금 미개통) |
| 실제 수익 | 자기보고 편차 극심: 2~3시간 $4.43~5.40 → 며칠 뒤 하루 종일 $0.43. floor 1.5h/일 ≈ 월 $6 |

**핵심 차이: kickbacks는 현금이고 실제 광고 매출의 50% 연동이다.** 매출이 없으면 지급도 없다(자기부담 아님).
클로애드 알파는 매출 0(무료 노출) → **리워드 100% 회사 자기부담**이므로 비현금·저단가 설계가 타당하다.

### 2.2 캐시워크 (한국 리워드 앱 대표)

| 항목 | 내용 |
|---|---|
| 만보기 일일 최대 | **100캐시 = 100원** (10,000걸음, 100걸음당 1캐시) |
| 1캐시 | 1원 |
| 추가 적립 | 돈버는퀴즈·럭키박스·광고 시청 등으로 하루 몇십~몇백원 |
| 초기화 | 매일 자정, 상자를 직접 탭해야 적립 |
| 첫 교환 | 만보기만으로 월 3,000원 ≈ 기프티콘 1장 |

## 3. 1,000명 리워드 부채 (매출 0, 전액 회사 부담)

노출/일 → 리워드/일 (× 20 활동일 = 월):

| 사용 강도 | 노출/일 | 리워드/일 | 월/인 | 1,000명 월 |
|---|---|---|---|---|
| 라이트 | ~120 | 36P | 720P | **72만원** |
| 미디엄(1h/일) | ~240 | 72P | 1,440P | **144만원** |
| 헤비(2h+/일) | ~480 | 144P | 2,880P | **288만원** |
| 상한 도달 | 500 | 150P | 3,000P | 300만원 |
| **절대 상한** (전원 매일 150P×30일) | 500 | 150P | 4,500P | **450만원** |

현실 혼합이면 **월 100만~200만원** 수준의 적립 부채.

## 4. 적립 부채 ≠ 실제 현금 지출

- 최소 교환 3,000P라서 라이트(월 720P)는 ~4개월, 미디엄(1,440P)은 ~2개월 모아야 첫 교환.
- **알파 1개월차 실제 현금 유출은 거의 0**, 미교환 이탈분은 소멸(breakage).
- **그러나 이것이 리텐션의 함정이다** — 비용은 낮지만 첫 보상까지 너무 오래 걸려 테스터가 이탈한다.

## 5. 결론: 액수가 아니라 습관·첫 보상이 리텐션을 만든다

| | 캐시워크 만보기 | 클로애드(현재) |
|---|---|---|
| 하루 상한 | 100원 | **150원** |
| 최소 교환 | 3,000원 | 3,000P |
| 상한 기준 첫 교환 | 30일 | **20일** |

- 클로애드 리워드 **액수는 이미 캐시워크보다 후하다.** 캐시워크는 하루 100원으로 수천만 명을 유지한다.
- kickbacks 실측도 월 $6, 최악 하루 $0.43 — 이 수준 금액은 리텐션 동인이 아니다.
- **리텐션 동인은 (1) 소액이라도 매일의 습관 루프, (2) 도달 가능한 첫 보상, (3) 쉬운 교환.**
- 캐시워크의 "매일 상자 탭"에 해당하는 능동 행동이 클로애드엔 없다(광고가 대기 중 수동 노출).

## 6. 알파 권장안 (총비용 거의 안 늘리면서 체감 개선)

1. **알파 한정 최소 교환 하향** (예: 3,000P → 1,000P) — 미디엄 유저가 2주 안에 첫 쿠폰.
   정책값이라 코드 수정 없이 `reward-policy.default.json`으로 변경 가능(CLAW-12 설계 의도).
2. **가입/마일스톤 보너스** — 첫 100회 노출 달성 시 소액 쿠폰. 1,000명 전원 1,000원권 1회 = 일회성 100만원(마케팅비).
3. **비현금 동기** — 리더보드·뱃지·얼리액세스. 알파는 "돈"보다 "만드는 데 참여"가 강한 동인.

**추천 알파 예산:** 리워드 부채 월 100만~200만원 + 웰컴 보너스 일회성 100만원.
첫 달 실제 현금 유출은 웰컴 보너스 위주 ~100만원, 이후 교환이 붙으며 월 100만원대 수렴.

## 7. 미확정 (선행 이슈)

- 세율·원천징수 → CLAW-13 / 전자금융거래법 → CLAW-14. 최종 단가·예산은 두 결론 반영 후 확정.
- 정책값(최소 교환·웰컴 보너스) 실제 조정은 별도 결정·PR로 진행한다.
