클로애드(ClawAd) 설치하기

Claude Code 상태줄에 [광고]로 표시된 광고를 제공하고, 서버가 검증한 노출만큼 포인트를 드리는 개발자 리워드 서비스입니다.

클로애드는 Anthropic·Claude와 제휴·후원 관계가 없는 독립 서비스입니다. 광고에는 항상 [광고] 표기가 붙습니다.

1. 준비물

  • Node.js 24 이상 — node --version으로 확인하세요.
  • Claude Code — 이미 설치되어 사용 중이어야 합니다.
  • 지원 OS: Windows · macOS · Linux

2. 설치

터미널에 아래 한 줄이면 됩니다. 저장소를 내려받거나 회원가입을 먼저 할 필요가 없습니다.

# macOS · Linux npx --yes <설치 패키지 URL> setup # Windows PowerShell npx.cmd --yes <설치 패키지 URL> setup
💡 지금은 폐쇄형 알파 기간입니다. <설치 패키지 URL>은 알파 테스터 초대 안내와 함께 개별 전달됩니다. 초대를 받지 않으셨다면 문의 창구로 참여 의사를 알려주세요.

3. 설치하면 일어나는 일

  1. 실행 환경을 진단합니다 (Node 버전, Claude 설정 쓰기 권한).
  2. 기존 상태줄(statusLine) 설정을 먼저 백업한 뒤 클로애드 상태줄을 설치합니다. 변경 내용은 설치 전에 화면에 고지됩니다.
  3. 광고를 미리 받아 두는 자동 동기화(sync)를 등록합니다 — 상태줄 표시는 네트워크 없이 로컬 캐시만 읽어 Claude Code를 느리게 하지 않습니다.
  4. Google·Kakao·Naver 소셜 로그인으로 계정을 연결합니다.

설치가 끝나면 Claude Code 상태줄에 이런 광고가 표시됩니다:

[광고] 예시 광고 문구가 이 자리에 표시됩니다

4. 포인트 적립과 교환

  • 광고 노출은 서버 검증을 거쳐 예상 적립 → 검증 중 → 확정 리워드 순서로 반영됩니다. 화면의 "예상" 수치는 확정값이 아닙니다.
  • 현재 적용 중인 적립 단가·일일 상한·최소 교환 기준은 리워드 샵 첫 화면에 항상 표시됩니다.
  • 확정 포인트는 리워드 샵에서 지정 상품(모바일 쿠폰)으로 교환합니다. 리워드는 비구매형·비양도형이며 충전·양도·현금 환급을 지원하지 않습니다.

5. 관리 명령

설치에 사용한 것과 같은 <설치 패키지 URL>을 그대로 씁니다.

… status설치 상태·로그인·동기화 상태를 확인합니다.
… pause광고 표시를 일시중지합니다. 포인트와 계정은 유지됩니다.
… resume일시중지한 광고 표시를 다시 켭니다.
… update체크섬 검증을 거쳐 새 버전으로 업데이트합니다. 실패하면 이전 버전으로 자동 복귀합니다.
… uninstall클로애드를 제거하고 백업해 둔 원래 상태줄 설정을 복원합니다.

예: npx --yes <설치 패키지 URL> pause

6. 어떤 정보를 보내나요?

보내는 것 (전부입니다): 노출 토큰, 노출 순번, 가명 기기식별자(로컬 생성 랜덤값), 광고 표시 시작·종료 시각, 표시 시작 신호, 회원 식별자, 클라이언트 버전.

절대 보내지 않는 것: 프롬프트, Claude 응답, 소스코드, 파일명·경로, Git 저장소명, 터미널 명령어, 환경변수, 클립보드, 접속 IP(이벤트 데이터로), MAC 주소 등 하드웨어 식별자. 클라이언트 코드가 이런 정보에 접근하지 못하도록 설계되어 있습니다.

자세한 내용: 개인정보처리방침

7. 자주 묻는 질문

Q. 광고를 잠깐 끄면 불이익이 있나요?
없습니다. 끄는 동안 포인트가 적립되지 않을 뿐입니다.

Q. 제거하면 원래 상태줄로 돌아가나요?
네. 설치 시 백업한 설정을 그대로 복원합니다. 계정과 이미 확정된 포인트는 유지되어 웹에서 교환할 수 있습니다.

Q. 여러 기기에서 써도 되나요?
됩니다. 등록 한도 내에서 기기를 추가할 수 있고, 포인트 상한은 기기가 아니라 계정 단위입니다. 같은 계정의 여러 기기에서 광고가 동시에 표시되면 한 건만 인정되지만, 이는 제재가 아니며 불이익이 없습니다.

Q. 포인트를 현금으로 바꿀 수 있나요?
아니요. 리워드는 지정 상품 교환 전용입니다.
